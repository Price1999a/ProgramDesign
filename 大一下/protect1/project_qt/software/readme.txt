网店管理系统用户手册
	1.引言
			该用户手册为可能使用该软件的人员所写。为大一下第一次项目。
	2.软件概述
		2.1目标
			该软件用于管理一个简单的网店系统。
		2.2功能
			该软件提供了用户注册、登陆，用户浏览库存，管理购物车文件，管理员管理库存、查看已购清单的功能。
		2.3性能
			该软件具备在较低时间延迟下运行的能力。
	3.运行环境
			该软件运行在64位Windows系统下，基于c++,QT5.9.2编写，使用msvc2017编译。
	4.使用说明
		4.1安装
			该软件为绿色软件，解压即可。
		4.2输入
			ID：5个字符，开头字符为大写字母，后4位为数字。名称：不超过20个字符，中文汉字或英文字母。品牌：不超过20个字符，中文汉字或英文字母。价格：大于零的小数，精确到小数点后一位。数量：非负整数（-1表示商品被删除）。用户名：不超过20个字符，只由小写字母和数字组成。整个库存文件不超过100行。整个已售清单不超过2000行。
		4.3出错求助
			请联系软件开发者。
		4.4功能列表
			用户登陆、用户注册、管理员登陆、管理员查询商品、管理员增加商品、管理员删除商品、管理员修改商品信息、管理员查看售货清单、用户查看商品、用户搜索商品、用户用户添加商品至购物车、用户删除购物车商品、用户查看购物车、用户结账。
